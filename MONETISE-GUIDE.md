# vinral — monetisation playbook (affiliate links, ads, Google dashboard)

Everything you asked for, step by step. Read the three honest notes first — they save you time and money.

1. **I can't generate your personal affiliate links.** Each link is tied to *your* affiliate account. So below I give you (a) where to find each product and (b) exactly how to mint your own link. You paste it into `products.js` → `link`.
2. **You can't reuse competitors' ad videos/images** — it's copyright + gets your ad account banned. Instead I show you the ad libraries to *study* what works, then make your own (your footage, supplier footage you're licensed to use, or UGC).
3. **Google Analytics tracks traffic & clicks, not payouts.** Payouts live in each affiliate dashboard (Amazon/AliExpress/Awin). The site already fires click events so GA can show you which products people click.

## PART 3 — How to mint each affiliate link

**Amazon EU Associates (SiteStripe):**
1. Get approved, then log in and browse to the product on amazon.de (or any EU store).
2. The **SiteStripe** bar appears at the top → click **Text** → copy the short link (amzn.to/…).
3. Paste it into `products.js` → that product's `link`.
4. Tip: use **OneLink** so a German link also earns from .fr/.it/.es visitors.

**AliExpress Portals:**
1. Log in at portals.aliexpress.com → **Tools → Link Generation**.
2. Paste the product URL → **Get tracking link** → copy it.
3. Paste into `products.js` → `link`.

**Awin (brand programs):**
1. Apply to the merchant inside Awin → once approved, open **Link Builder**.
2. Paste the product/brand page URL → generate → copy → paste into `products.js`.

Then in `products.js`, for each product change:
`link: "#REPLACE-WITH-AFFILIATE-LINK"` → `link: "https://your-real-link"`. Save, refresh.

## PART 4 — Ads: research, create, publish

**Study what's working (do NOT copy):**
- **TikTok Creative Center** (free): https://ads.tiktok.com/business/creativecenter/ — Top Ads + Trends, filter by country (DE/AT/NL).
- **Meta Ad Library** (free): https://www.facebook.com/ads/library/ — search a product/brand to see every ad they're running on FB/IG.
- Optional paid spy tools: Minea, PiPiAds, Dropispy.

**Make your own creative:** film the product yourself, use **supplier video you're licensed to use** (ask the AliExpress seller / brand), or commission UGC. Hook in the first 2 seconds, vertical 9:16, captions on, show the before/after or the satisfying moment. Each product's angle is in `products.js` → `adAngle`, and the Top-30 sheet.

**Where to publish (start free, then pay behind winners):**
- TikTok — organic posts first, then **Spark Ads** to boost a video that already performs.
- Instagram **Reels** + Facebook — via Meta Ads Manager once you have a winner.
- YouTube **Shorts** — repost the same vertical clips.
- Put your site link (or `vinral.com/?p=neck-fan` for a specific product) in bio / ad destination.

## PART 5 — Google dashboard (GA4) to see clicks & activity

1. Go to https://analytics.google.com → **Admin → Create → Account** (name it vinral) → **Create property**.
2. Add a **Web data stream** for `https://vinral.com` → copy the **Measurement ID** (looks like `G-XXXXXXXXXX`).
3. Open `index.html`, find the **GOOGLE ANALYTICS (GA4)** block in the `<head>`, paste your ID in **both** spots, and remove the `<!--` and `-->` around it.
4. **EU consent:** GA uses cookies, so add a small consent banner first and only load GA after the user accepts (free: Osano CookieConsent or Klaro). 
5. The site already fires two events: `select_item` (a product is viewed) and `outbound_click` (someone clicks through to a store). See them in **GA4 → Reports → Engagement → Events**.
6. Mark `outbound_click` as a **Key event** (GA4 → Admin → Events) so it counts as a conversion.
7. **Payouts:** not in GA — check Amazon Associates / AliExpress Portals / Awin dashboards. (Later we can build one simple page that lists clicks next to payouts.)
